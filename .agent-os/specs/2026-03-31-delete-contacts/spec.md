# Requirement - Delete Contacts

In the Activator Contacts and Chaser Contacts tables add the following feature. This will allow users to delete one or more
contacts.

 - add a checkbox at the end of each contact. 
 - at the top of the table add a two buttons: Select All/Select None
 - at the top of the table add a dropdown menu with one option: Delete.
 - at the top of the table add an Apply button

When the user presses the Apply button a confirmation dialog
should be shown with a count of the number of contacts to be deleted.

When confirmed the contacts should be deleted from the database.

Use your UI superpower to design the UI to follow best practices.
Create an implementation plan in this directory before proceeding.
